### Koncept

FlashSale One-Product App - aplikacja w stylu drop, która sprzedaje tylko jeden limitowany produkt dnia.

### Flow aplikacji

Użytkownik wchodzi do aplikacji, widzi licznik czasu oraz jeden produkt. Wciska przycisk kup teraz.

Produkt -> Koszyk -> Płatność

### Uruchomienie

1. Wejść do katalogu /DropIt/Backend
2. W konsoli wpisać "npm run server"
3. Aby wyświetlić aplikację, w przeglądarce należy wpisać http://localhost:5000
